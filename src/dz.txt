ДЗ


1. Банк — результат перевода как объект


Условие

Есть класс BankAccount:


class BankAccount {
    String owner;
    double balance;

    BankAccount(String owner, double balance) {
        this.owner = owner;
        this.balance = balance;
    }
}

Создай класс TransferResult:


class TransferResult {
    boolean success;
    String message;
    double remainingBalance;
}


Задание

В классе Bank реализуй метод:

TransferResult transfer(BankAccount from, BankAccount to, double amount)


Метод должен:

если денег хватает:

списать деньги

зачислить деньги

вернуть объект TransferResult с success = true

если не хватает:

ничего не менять

вернуть объект TransferResult с success = false

В main:

создай два аккаунта

вызови transfer

используй возвращённый объект:

выведи сообщение

если success == true, выведи новый баланс
