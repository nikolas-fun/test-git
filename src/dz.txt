1. Уникальные слова в предложении
В main есть строка:
String sentence = "java is fun and java is powerful";

Требования:
Разбей строку на слова.
Сохрани все слова в HashSet<String>.

Выведи:
количество уникальных слов
сами уникальные слова

После ввода — вывести:
сколько уникальных номеров получилось
