1. Подсчёт элементов (ArrayList)

Условие:
Есть ArrayList<String> со словами.

Нужно:

Оставить слова длиной больше 5

Посчитать количество таких слов

Пример входных данных:

["stream", "java", "collection", "api", "filter"]


Ожидаемый результат:

2

Подсказка:
filter → count

2. Проверка условий (HashSet)

Условие:
Есть HashSet<String> с email-адресами.

Нужно:

Проверить, все ли email содержат символ @

Проверить, есть ли хотя бы один email с доменом "gmail.com"

Пример входных данных:

["test@gmail.com", "admin@yahoo.com", "user@mail.com"]


Ожидаемый результат:

allMatch → true
anyMatch → true


Подсказка:
allMatch, anyMatch
